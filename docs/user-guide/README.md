# Roller Rumble — Operator Wiki

This is the plain-language handbook for running **Roller Rumble**, the app that powers our
live stationary-bike race events. It's written for the people actually running the show at an
event — no programming background needed.

## Who this is for

If you are setting up the laptop before an event, adding racers, starting races, running a
tournament, or trying to figure out why something isn't working, you're in the right place.

## Two ways to use this handbook

1. **Read it directly.** Each page is a step-by-step guide for one part of the app. Start with
   the Overview below, then jump to whatever you need.
2. **Ask an AI assistant.** You can paste one of these pages (or several) into a chat with an AI
   assistant like Claude or ChatGPT and ask it questions in your own words — for example,
   _"The racer page won't let someone sign in, what do I check?"_ The pages are written so an AI
   can give you accurate answers from them. For best results, paste the specific page your
   question is about **plus** the Troubleshooting page.

## A note on "test mode" vs. "the real thing"

The app can run with a **simulator** that pretends a bike is being pedaled, so you can practice
the whole flow without hardware. Real race hardware support is still being finished. When a page
mentions the "sensor" or "simulator," that's what it means. Everything in this handbook works the
same whether you're practicing with the simulator or running a live event.
